# sunny-datko

My personal portfolio — built with React, Vite, TypeScript, and Material UI. Check it out live at [sunnydatko.com](https://sunnydatko.com).

## Running locally

```bash
git clone https://github.com/sunnydatko/sunny-datko.git
cd sunny-datko
npm install
npm run dev
```

The dev server runs at http://localhost:5173.

## Tech stack

- **React** — UI
- **Vite** — build tooling
- **TypeScript** — type safety
- **Material UI** — component library and styling
